# Imagine Hero - Fundal Pagina Principală

Acest folder conține imaginea de fundal pentru secțiunea Hero de pe pagina principală.

## Cum să adaugi imaginea

1. **Locația**: Adaugă imaginea direct în acest folder (`public/images/hero/`)

2. **Numele fișierului**: 
   - **`hero-background.jpg`** (sau `hero-background.png`)

3. **Format recomandat**: 
   - Format: JPG sau PNG
   - Dimensiune recomandată: 1920x1080px sau mai mare (16:9 aspect ratio)
   - Calitate: imagini de bună calitate, optimizate pentru web
   - Orientare: orizontală (landscape)

## Recomandări pentru imagine:

- **Conținut sugerat**: 
  - Atelier auto, mașini în service
  - Echipamente de service
  - Zona de lucru profesională
  - Mașini în reparație
  - Atmosferă profesională de service auto

- **Caracteristici tehnice**:
  - Aspect ratio: 16:9 (ex: 1920x1080px, 2560x1440px)
  - Format: JPG (recomandat pentru fotografii) sau PNG
  - Dimensiune optimă: 1920x1080px pentru cea mai bună calitate
  - Calitate: optimizată pentru web (max 500KB-1MB recomandat)
  - Orientare: orizontală (landscape)

## Notă importantă:

- Imaginea va fi afișată cu opacitate 30% pentru a nu acoperi conținutul
- Dacă imaginea nu există, site-ul va folosi gradient-ul de fundal existent
- Poți folosi și formatul PNG dacă preferi (ex: `hero-background.png`)
- Asigură-te că imaginea este optimizată pentru web pentru performanță bună

## Optimizare imagini:

Pentru cea mai bună performanță, recomandăm:
- Comprimare imagini înainte de upload
- Folosirea formatului JPG pentru fotografii
- Dimensiune recomandată: 1920x1080px pentru calitate bună
- Aspect ratio: 16:9 pentru consistență
