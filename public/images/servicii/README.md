# Imagini Servicii

Acest folder conține imaginile pentru paginile individuale ale serviciilor.

## Cum să adaugi imaginile

1. **Locația**: Adaugă imaginile direct în acest folder (`public/images/servicii/`)

2. **Format recomandat**: 
   - Format: JPG sau PNG
   - Dimensiune recomandată: 800x600px sau 1200x800px (raport 4:3 sau 3:2)
   - Calitate: imagini de bună calitate, optimizate pentru web

3. **Numele fișierelor**: Trebuie să folosești EXACT aceste nume (cu litere mici și cratime):

## Lista completă de nume de fișiere:

- `diagnoza.jpg` - pentru serviciul de Diagnoză
- `kit-ambreiaj.jpg` - pentru Schimbare Kit Ambreiaj
- `kit-distributie.jpg` - pentru Înlocuire Kit-uri de Distribuție
- `tren-rulare.jpg` - pentru Înlocuire Subansamble Tren de Rulare
- `sistem-franare.jpg` - pentru Verificare și Reparații Sistem de Frânare
- `reparatii-motoare.jpg` - pentru Înlocuiri și Reparații Motoare
- `schimb-ulei-filtre.jpg` - pentru Schimb Ulei / Filtre
- `vulcanizare.jpg` - pentru Vulcanizare (general)
- `geometrie-3d.jpg` - pentru Geometrie 3D
- `echilibrare.jpg` - pentru Echilibrare
- `indreptare-roluri-jante.jpg` - pentru Îndreptare Roluri Jante
- `dejantat.jpg` - pentru Dejantat

## Notă importantă:

- Dacă o imagine nu există, site-ul va afișa automat o ilustrație SVG
- Poți folosi și formatul PNG dacă preferi (ex: `diagnoza.png`)
- Asigură-te că imaginile sunt optimizate pentru web (nu prea mari, max 500KB recomandat)

## Optimizare imagini:

Pentru cea mai bună performanță, recomandăm:
- Comprimare imagini înainte de upload
- Folosirea formatului JPG pentru fotografii
- Folosirea formatului PNG doar dacă ai transparență
- Dimensiune maximă: 1920x1080px pentru calitate bună
